GitHub Pages용 정적 포트폴리오 사이트입니다.

## 구성

- `index.html` — 메인 포트폴리오 페이지
- `styles.css` — 디자인 스타일

## GitHub Pages 배포 방법

1. `juyoungjudylee/portfolio` 저장소에 위 파일들을 업로드합니다.
2. GitHub 저장소에서 **Settings → Pages**로 이동합니다.
3. **Build and deployment → Source**를 `Deploy from a branch`로 설정합니다.
4. Branch는 `main`, folder는 `/root`로 설정합니다.
5. 저장 후 발급되는 Pages URL을 확인합니다.
